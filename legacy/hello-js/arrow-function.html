  • Стрелочные функции

    Еще один вариант синтаксиса функций, это функции стрелки.
    let func = (arg1, arg2, ...) => expression

    Здесь создается функция с параметрами arg1 и arg2, вычисляется тело функции expression и результат вычислений возвращается. Если параметр один, то круглые скобки опускаются, а если их нет, то наоборот остаются только они. Есть возможность создавать и многострочные стрелочные функции, однако тогда нужно использовать return, если это необходимо.
