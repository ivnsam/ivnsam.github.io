Назад

Операторы сравнения, как и другие операторы, возвращают некоторое значение. В данном случае возвращаемое значение имеет лоигческий тип

  • Сравнение строк

    1. Сравниваются первые символы строк.
    2. Если первый символ первой строки больше (меньше), чем первый символ второй, то первая строка больше (меньше) второй.
    3. Если первые символы равны, то таким же образом сравниваются уже вторые символы строк.
    4. Сравнение продолжается, пока не закончится одна из строк.
    5. Если обе строки заканчиваются одновременно, то они равны. Иначе, большей считается более длинная строка.

  • Сравнение разных типов

    При сравнении разных типов, оба сравниваемых операнда приводятся к числу, но в данном случае "0" становится 0, в то время как Boolean("0") вернет true, т.е. 1

  • Строгое сравнение

    Обычное сравнение не отличит 0, false и пустую строку "", из-за этого могут возникнуть проблемы
    Для того, чтобы данные проблемы избежать, используется оператор строгого сравнения "==="
    Оператор строгого сравнения не приводит типы сравниваемых операнд, поэтому, если сравниваются разные типы, то результатом сравнения будет false

  • Сравнение null с undefined

    1. При строгом сравнении (===) результатом будет false, так как это разные типы
    2. При нестрогом сравнении (==) результатом будет true. Эти два значения равны между собой и не равны больше ничему. Это специальное правило языка
    3. При использовании математических операторов и других операторов сравнения (<, >, <=, >=) null и undefined преобразуются в числа 0 и NaN соответственно

    Так как null и undefined имеют своеобразные правила сравнения, то лучше делать отдельную проверку на эти два значения
