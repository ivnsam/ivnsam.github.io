  • Функция-конструктор

    Функции-конструкторы являются обычными функциями, с помощью которых можно создавать большое количество однотипных объектов. Существует два соглашения:
    1. Имя функции-конструктора должно начинаться с большой буквы;
    2. Функция-конструктор должна вызываться при помощи оператора new.
    function User(name) {
    this.name = name;
    this.isAdmin = false;
    }

    let user = new User("John");

    alert(user.name); //John
    alsert(user.isAdmin); //false

    Когда функция вызывается как new User(...), происходи стледующее:
    1. Создается новый пустой объект и он присваивается this;
    2. Выполняется код функции;
    3. Возвращается значение this.

    Помимо свойств в конструкторе объекта можно добавлять и методы объекта. function User(name) {
    this.name = name;

    this.sayHi = function() {
    alert( "My name is " + this.name );
    };

    let user = new User("User");
    user.sayHi(); //My name is User
    }
