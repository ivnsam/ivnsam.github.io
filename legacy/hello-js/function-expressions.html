  • Функции-выражения

    Синтаксис
    let имя = function() {
    ...тело...
    };
    let some = имя;
    имя();
    some();
    Здесь фукнция создается явно и ей, как переменной, присваивается значение. В конце объявления такой переменной необходима точка с запятой (;).
    В JavaScript все функции это значения, поэтому их можно присваивать, выводить как и переменные. Можно даже скопировать функцию, а затем вызвать ее, работать будут обе.

  • Функции-коллбэки

    Синтаксис function ask(question, no, yes) {
    if (confirm(question)) yes()
    else no();
    }
    function showOK() {
    alert('OK');
    }
    function showCANCEL() {
    alert('CANCEL');
    }
    ask('ok?', showCANCEL, showOK);

    Здесь функция задаст вопрос и в зависимости от ответа выполнится первая или вторая функция, переданная при вызове ask('ok?', showCANCEL, showOK). Аргументы в такой функции называются колбэк-функциями или просто колбэками. Основная идея заключается в том, что мы передаем функцию в аргументе и ожидаем, что когда-нибудь она вызовется обратно.

    Этот пример можно переписать еще короче, используя анонимные функции.
    function ask(question, no, yes) {
    if (confirm(question)) yes()
    else no();
    }
    ask(
    'ok?',
    function() {alert('OK'); },
    function() {alert('CANCEL'); }
    )
    Здесь анонимными функции объявляются прямо в вызове ask() и не имеют названия, поэтому и называются анонимными.
