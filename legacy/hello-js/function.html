  • Переменные

    Локальная переменная Если переменная объявлена внутри функции, то видна она будет только внутри этой функции.
    Внешняя переменная Если переменная объявлена вне функции, то видна она будет как внутри, так и вне этой функции. Функция можте изменять внешнюю переменную. При совпадении имен локальной и внешней переменной будет использована локальная переменная.

  • Параметры

    Параметры, передаваемые в функцию могут так же называться аргументами функции. Переданные таким образом значения копируются в одноименные локальные переменные, тем самым внешние переменные не изменяются.
    Значение по умолчанию (или функция someFunction(), указанная вместо него) в объявлении функции будет использоваться, если при вызове функции не передан данный параметр.

  • Возврат значения

    Директива return служит для остановки выполнения тела функции и возвращения некоторого значения в вызвавший данную функцию код. Если не задано какое значение возвращать, то будет возвращено undefined

  • Функции и комментарии

    Одна функция всегда должна выполнять одно действие. Для простоты чтения кода лучше делать несколько небольших функций, чем одну делающую все сразу. function showPrimes(n) { for (let i = 2; i < n; i++) { if (!isPrime(i)) continue; alert(i); // простое } } function isPrime(n) { for (let i = 2; i < n; i++) { if ( n % i == 0) return false; } return true; } Здесь вместо того, чтобы объединить эти две функции и сделать все в одной, лучше сделать так как показано выше. В данном случае название функции isPrime() уже подсказывает что делает эта функция. Такой код еще называют самодокументируемым
