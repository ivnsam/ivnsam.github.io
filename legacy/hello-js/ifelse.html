  • Оператор if-else

    Оператор if() выполняет команду
    year = 2007;
    if (year == 2007) alert("2007 year back!");
    или блок кода
    year = 2007;
    if (year == 2007) {
    alert("2007 year back!");
    alert("да здравствует машина времени!");
    }
    когда в скобках значение true

    Блок else - это необязательный блок, код в котором будет выполнен, если условие в скобках if () ложно
    Для задания нескольких условий можно использовать конструкцию else if ()

  • Условный оператор "?"

    Данный оператор служит для проверки условия при присвоении переменной некоторого значения
    let result = условие ? значение1 : значение2;
    let accessAllowed = (age>21) ? true : false;
    Если условие верно, присваивается значение1, иначе значение2
    Оператор "?" можно использовать и как замену if, однако читаемость кода при этом ухудшается, поэтому делать этого не рекомендуется
