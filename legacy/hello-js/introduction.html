Подключение скриптов

  • Можно написать скрипт прямо в html коде в тэге: <script></script>
  • Можно подключить как отдельынй файл скрипта: <script src="/path/to/script.js"></script> Указывают или абсолютный или относительный путь до скрипта

В начале каждого скрипта надо писать "use strict" или 'use strict'. Эта директива включает "современный" или "строгий" режим. Писать use strict, чтобы он заработал, нужно только в первой строке. Над ним могут быть только комментарии.

Переменные

Переменные создаются с помощью ключевого слова let
let message;
После создания можно присвоить переменной значение
message="some text"
Вывести значение переменно можно, например, так
alert(message);

Имена переменных

Имена переменных могут содержать только буквы, цифры, символы $ и _
Первый символ не должен быть цифрой
Если имя содержит несколько слов, используется верблюжья нотация, например veryLongNameOfVariable
Названия переменных чувствительны к регистру. variable и Variable это две разных переменных
Существует список зарезервированных слов, которые использовать нельзя

Константы

Константы создаются с помощью ключевого слова const
Имена в верхнем регистре принято давать тем константам, которые известны до начала исполнения скрипта, но трудно запоминаемы
const COLOR_ORANGE="#FF7F00";
