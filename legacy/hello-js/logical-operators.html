  • || оператор ИЛИ

    Обычно оператор ИЛИ используется для манипулирования булевыми значениями
    Таблица истинности

    a b a || b
    0 0 0
    1 0 1
    0 1 1
    1 1 1
    Если значение не логического типа, оно к нему приводится, однако возвращено будет значение, а не результат преобразования, которое первым оказалось правдивым (true), если такого не оказалось, будет возвращено последнее значение в выражении
    Этой особенностью можно воспользоваться для нахождения первого истинного значения
    result = value1 || value2 || value3;
    Также это позволяет делать сокращенные вычисления, т.е. вычислять значение какого-либо выражения, только при определенных условиях
    let x;
    true || (x = 22+9);
    alert(x); //undefined
    false || (x = 22+9);
    alert(x); //31

  • && (оператор И)

    Оператор И используется аналогично ИЛИ
    Таблица истинности

    a b a && b
    0 0 0
    1 0 0
    0 1 0
    1 1 1
    Если значение не логического типа, оно к нему приводится, однако возвращено будет значение, а не результат преобразования, которое первым оказалось ложным (false), если такого не оказалось, будет возвращено последнее значение в выражении
    Этой особенностью можно воспользоваться для нахождения первого ложного значения

  • ! (оператор НЕ)

    Оператор приводит аргумент к логическому типу, а затем возвращает его обратное значение
    result = !value;
    Двойное НЕ можно использовать вместо Boolean()
    result = !!value;
