  • Методы объекта

    Чтобы объявить метод объекта, т.е. то, что объект "умеет" делать, есть два способа. Первый - это использовать Function Expression чтобы создать функцию и присвоить ее свойству объекта. Второй - заранее объявить функцию, а затем использовать ее в качестве метода.
    Синтаксис через Function Expression.
    let user = {
    name: "John",
    age: 24,
    };

    user.sayHi = function() {
    alert("Hi!");
    };

    user.sayHi();

    Синтаксис через использование отдельной функции в качестве метода.
    let user = {
    //...
    };

    //сначала объявление
    function sayHi() {
    alert("Hi!");
    };

    //затем добавление в качестве метода
    user.sayHi = sayHi;

    //теперь использование
    user.sayHi();

    Существует сокращенный синтаксис объявления метода объекта через Function Expression. Хоть эти две записи не совсем эквивалентны, зачастую предпочтительнее использовать сокрщенную форму.
    let user = {
    name: "John",
    age: 24,
    sayHi() {
    alert("Hi!");
    },
    };

  • Ключевое слово "this" в методах

    Данное ключевое слово используется методом для доступа к свойству внутри объекта.
    let user = {
    name: "John",
    age: 24,

    sayHi() {
    //"this" - это текущий объект
    alert(this.name);
    },
    };

    user.sayHi(); // John

    "this" не является фиксированным, т.е. значение this вычисляется непосредственно во время выполнения кода и зависит от контекста.
    let user = {name: "User",};
    let admin = {name: "Admin",};

    function sayName() {
    alert(this.name);
    };

    user.func = sayHi;
    admin.func = sayHi;

    //в данном случае this в каждой строке имеют разные значения
    user.func(); //User, this = user
    admin.func(); //Admin, this = admin

    //не важно как вызывается метод, this работает и при вызове через скобки
    admin["func"];// Admin

    При использовании this в стрелочных функциях его значение берется из внешней функции.
