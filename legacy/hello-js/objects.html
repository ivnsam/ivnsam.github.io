Назад

  • Создание объекта

    Объект может быть создан двумя способами: с помощью фигруных скобок (литерал объекта/литеральная нотация) или с помощью конструктора объекта.
    let user = new Object(); //конструктор объекта
    let user = {}; //литерал объекта
    Объекты содержат в себе свойства. Свойства это пара "ключ:значение", где ключ - это строка (также ее называют "именем свойства"), а значение может быть чем угодно. При использовании литерального синтаксиса можно сразу задать свойства объекта.
    let user = {
    name: "John",
    age: 30
    };

    Действия над свойствами объектов

    Для обращения к свояствам используют запись "через точку" либо через квадратные скобки.
    alert( user.name );
    alert( user["name"] );
    Для добавления свойств и удаления свойств используется следующий синтаксис:
    let user = {
    name: "John",
    age: 30
    };
    user.isAdmin = true; //добавление свойства через точку
    delete user.age; //удаление свойства через точку
    user["isAdmin"] = true; //добавление свойства через квадратные скобки
    delete user["age"]; //удаление свойства через квадратные скобки
    На имена свойств нет никаких ограничений за исключением имени __proto__. Имя свойства может состоять из нескольких слов, но тогда обращение должно производиться через квадратные скобки
    let user = {
    name: "John",
    age: 30,
    "likes birds": true,
    };
    Последнее свойство в объявлении переменной может заканчиваться запятой. Это называется "висячая запятая". Это упрощает добавление, перемещение и удаление свойств, так как все строки становятся одинаковыми.

    Вычисляемые свойства

    Еще один вариант обращения к свойству через переменную, которая может быть до этого вычислена, введена пользователем или задана изначально. Однако такое обращение возможно только через квадратные скобки, обращение через точку такого сделать не позволяет.
    let key = "likes birds";
    user[key] = true;
    //более сложные выражения
    let fruit = "orange";
    let bag = {
    [fruit + "Pi"]: 5, //bag.orangePi = 5
    };

    Свойства из переменной

    Бывает необходимо создать свойство, имя которого совпадает с именем переменной, значение которой надо сохранить в этом свойстве. Для этого существуют короткие свойства, которые можно совмещать с обычными свойствами в одном объекте.
    let user = {
    isAdmin: isAdmin, //создание свойства из переменной
    name, //создание короткого свойства из переменной (название свойства совпадает с именем переменной)
    age: 24, //обычное создание свойства
    };

    Проверка существования свойства

    Проверить существование свойства в JavaScript возможно двумя разными способами. Первый возможен благодаря тому, что при обащении даже к несуществующему свойству не произойдет ошибки, а будет возвращено значение undefined.
    let user{};
    alert( user.undefinedProperty === undefined ); //true - свойство не существует
    Второй способ реализован через специальный оператор "in", который специально создан для проверки существования свойства в объекте.
    let user = {
    name: "John",
    age: 30,
    };
    alert( "age" in user ); //true
    alert( "blahblahblah" in user ); //false
    Слово слева от "in" должно быть строкой в кавычках, либо переменной, тогда кавычки не пишутся.

    Цикл "for...in"

    Цикл "for...in" используется для перебора всех свойств объекта.
    for (key in object) {
    //тело цикла с действиями для каждого свойства объекта
    };
    Свойства в объекте упорядочены особым образом. Если ключи у свойств целочисленные, то они упорядочиваются по возрастанию, а если нет, то в том порядке, в котором они были добавлены в свойство.
