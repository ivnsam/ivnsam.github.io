Операнд - то, к чему применяется оператор. Нарпимер, 2+5, 2 и 5 это операнды
Унарный оператор - такой оператор, который применяется к одному операнду
Бинарный оператор - такой оператор, который применяется к двум операндам

  • Сложение строк, бинарный +

    Обычно "+" складывает числа, но если из двух операнд хотя бы один является строкой, то второй, если он не строка, будет преобразован к строке
    alert(1+"2"); //"12"
    alert("5"+3); //"53"
    alert(2+3+"9"); //"59", а не "239"

  • Преобразование к числу, унарный +

    Если применить "+" к одному операнду, то произойдет преобразование этого операнда в число, т.е. выполнится Number(value)
    В выражении преобразование выглядит так
    let a = "2";
    let b = "3";
    alert(+a + +b); //5

  • Приоритет операторов

    Приоритет операторов определяет, в какой последовательности будет выполнено то или иное действие
    Запомнить следует, что у унарных операторов приоритет выше, чем у бинарных. Самый низкий приоритет у оператора запятая
    Полная таблица приоритетов здесь

  • Присваивание

    "=" это тоже оператор с одним из самых низких приоритетов, поэтому сперва выполняется вся арифметика, а только после этого уже присваивание
    Возможно присваивание по цепочке a = b = c = 1 + 3; В данном случае все действия выполняются справа, тоесть сперва сложение, а затем присваивание c, b, a
    Оператор "=" возвращает значение, которое он только что присвоил, поэтому его можно использовать в более сложных выражениях
    let a = 1;
    let b = 2;
    let c = 3 - (a = b + 1);
    alert(a); // 3
    alert(c); // 0

  • Остаток от деления %

    Оператор % находит остаток от деления одного числа на другое

  • Возведение в степень **

    a ** b; возводит число a в степень b. Работает так же и для дробных чисел

  • Инкремент, декремент

    Инкремент увеличивает число на 1, декремент - уменьшает
    Применять эти операторы можно только на переменные, при применении к целым числам возникнет ошибка
    Данные операторы бывают двух форм, префиксной и постфиксной
    Префиксная форма возвращает новое (измененное) значение, постфиксная - старое (неизмененное)
    let counter1 = 0;
    counter2 = counter1;
    let pre = ++counter1;
    alert(pre); //1
    let post = counter2++;
    alert(post); //0

  • Побитовые операторы

    Побитовые операторы работают с 32-х разрядными целыми числами и приводят к ним при необходимости
    Подробнее о них можно почитать здесь

  • Сокращенная арифметика с присваиванием

    Если нужно применить оператор к переменной и сохранить результат в ней же, то можно использовать данный вид записи
    Например, вместо такой записи
    let n = 2;
    n = n + 1;
    n = n * 2;
    можно использовать такую
    let n = 2;
    n += 1;
    n *= 2;
    Таким образом можно использовать все арифметические и побитовые операторы

  • Оператор запятая

    Данный оператор разделяет несколько действий, но возвращает результат последнего
    for (a=1, b=3, c=a*b; a < 10; a++) {...}
