Назад

  • Строковое преобразование

    Все представляется в виде строки
    Конструкция String(value); преобразует значение value к строковому типу
    Так false станет "false", а null станет "null"

  • Численное преобразование

    Происходит в математических функциях и выражениях, например при делении
    alert("6" / "2"); в результате выведет 3, так как все преобразовалось к числовому типу
    alert(4+5+"px"); в результате выведет "9px", так как первым действием произвелось сложение числе, а затем конкатенация со строкой
    Явное преобразование выглядит так Number(value); Правила преобразования следующие

    Значение Результат преобразования
    undefined NaN
    null 0
    true / false 1 / 0
    string Пробелы по края строки обрезаются, далее, если строка пустая, то 0, если там есть число, то число, если там есть не число/не только число, то NaN
    Сложение объединяет строки, если в выражении есть хотя бы одно слагаемое в строковом виде
    alert("1" + 2); выведет 12, так как все преобразуется к строке

  • Логическое преобразование

    Преобразует все к логическому типу
    Явное преобразование делается так: Boolean(value);
    Правила преобразования следующие: 0, пустая строка, null, undefined и NaN преобразуются в false, остальное в true
    Однако строка "0" превратится в true, так же как и строка с пробелом " ", потому что эти строки считаются не пустыми
