Глубокое копирование

Функция copyObject(obj) произоводит глубокое копирование объекта и возвращает копию данного объекта. Также в скрипте есть функция isObject(val), которая с помощью Object.prototype.toString.call() проверяет переданное ей значение на то, объект ли это. Кнопка "Пример выполнения скрипта" выполняет функцию deepCopy(), которая служит для примера и выводит в консоль оригинальный объект и его копию.
